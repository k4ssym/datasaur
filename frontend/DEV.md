# Разработка фронтенда — автоматическое обновление

## Вариант A: Мгновенные правки без перезагрузки (рекомендуется)

1. Запустите бэкенд в одном терминале (из `my-project`):
   ```bash
   py -m uvicorn src.main:app --host 127.0.0.1 --port 8080 --reload
   ```
2. В другом терминале запустите фронтенд из папки `frontend`:
   ```bash
   npm run dev
   ```
3. Откройте в браузере **http://localhost:5173** (не 8080).
4. Сохраняйте файлы — изменения подхватываются сразу (HMR), без ручного билда и обновления страницы.

API-запросы идут на `http://127.0.0.1:8080` (бэкенд). CORS на бэкенде разрешён.

---

## Вариант B: Один адрес 8080 (билд копируется сам)

1. Запустите бэкенд:
   ```bash
   cd my-project
   py -m uvicorn src.main:app --host 127.0.0.1 --port 8080 --reload
   ```
2. В другом терминале из папки `frontend`:
   ```bash
   npm run build:watch
   ```
3. Откройте **http://localhost:8080**.
4. После каждого сохранения файла билд пересобирается и автоматически копируется в `my-project/frontend/dist`. Обновите страницу (F5 или Ctrl+R) — увидите изменения.

Один раз запустили `build:watch` — дальше только сохраняете код и обновляете вкладку.
